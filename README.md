# 逆勢補倉計算器

A real-time futures trading calculator for counter-trend add-on positions.

## Features

- **Live OKX price feed** — updates every second
- **Instant calculation** — type two prices, get results immediately
- **Range table** — see 16 add-on price scenarios at once
- **Multi-coin** — ETH, BTC, SOL, XRP, BNB
- **Long & Short** — works for both directions
- **No backend** — single HTML file, open in any browser

## Usage

Open `index.html` in your browser. No installation or server required.
